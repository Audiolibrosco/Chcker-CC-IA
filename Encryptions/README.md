# Gateway Encryptions

This is an encryptor for gateways.

## Setup Instructions

To run this project perfectly, follow the steps below:

1. Open the file:
  src/db/connection.js
2. Replace `<MONGO URL>` with your actual Database URL:
```javascript
await mongoose.connect('<MONGO URL>/SiteDB', {});
```
